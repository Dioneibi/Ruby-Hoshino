poner uno por uno 

```bash
termux-setup-storage
```

```bash
apt update && apt upgrade && pkg install -y git nodejs ffmpeg imagemagick yarn
```

```bash
git clone https://github.com/Dioneibi/Tatsumaki-Bot-MD && cd Tatsumaki-Bot-MD
```
```bash
yarn install && npm install
```
```bash
npm start
```

